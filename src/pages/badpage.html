Bad Web Design: DoleKemp '96

Visit Example Site

If you go back to 1996, you’ll arguably find the worst-designed website! The uncentered text and the wonky font size make the website unappealing to be on. Some of the links, although look clickable, are not designed to be clicked. Which leads to users being frustrated. The website is not nearly user-friendly, and the website is not able to easily maneuver through its sections, which probably led to a loss in votes. For a campaign website to be effective, especially in the modern day, the design must be easy to access, visually appealing, and easily maneuverable. Maybe that’s why he didn’t win his campaign!
